# customer
The components in this folder are for use by a user of type customer and type customer only

## CustomerUnlockBike
This component is responsible for displaying the unlock page for a customer that will display a drop down with all their trips available to unlock

## MyTrips
This component uses ListOfLiveTrips to display the current users list of trips. User can filter trips, make reports, cancel and unlock bikes ready to be unlocked

## Report
This component allows user to issue a report to be sent to an Operator by choosing a category, reservation ID and leaving a comment
